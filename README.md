# 🌿 Klarity — AI-Powered Email Simplifier

Klarity is an AI-based backend system that simplifies complex emails into easy-to-understand summaries. It is designed especially for users who face difficulty reading long or complicated text, including individuals with dyslexia.

---

## 🚀 Features

* 📧 Fetch latest emails using secure authentication
* 🧠 AI-powered text simplification (ASI1 model)
* 📊 Readability analysis using multiple metrics
* ✨ Highlights key points for quick understanding
* 🗺️ Word mapping (explains difficult words in simple terms)
* 🔐 Secure API handling with environment variables

---

## 🛠️ Tech Stack

* **Backend:** FastAPI
* **AI Model:** ASI1 (AI21 Labs / Jamba-based models)
* **Email Integration:** Composio (OAuth-based Gmail access)
* **Readability Analysis:** textstat
* **Language:** Python
